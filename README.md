# AI

🦿 Some ipython notebooks implementing AI algorithms

## 🔩 Getting Started

```shell
$ git clone git@github.com:1chooo/ai.git
$ cd ai
$ python3.11 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ deactivate
$ rm -rf venv     # remove the venv
```
